## 🎬 动画速查 (Animations)

### 动画时长 (Duration)

| 名称 | 时长 | 用途 |
|------|------|------|
| `instant` | 0ms | 无动画 |
| `fast` | 150ms | 快速响应 |
| `normal` | 300ms | 标准动画 ⭐ |
| `slow` | 500ms | 慢速动画 |
| `slower` | 800ms | 更慢动画 |
| `breathing` | 2500ms | 呼吸灯周期 |

### 缓动曲线 (Easing)

| 名称 | 用途 |
|------|------|
| `Ease In` | 淡入、缩小 |
| `Ease Out` | 淡出、放大 ⭐ |
| `Ease In Out` | 平滑过渡 ⭐ |
| `Ease Out Cubic` | 生命值变化 |
| `Elastic Out` | 骰子反弹 |
| `Bounce Out` | 弹跳效果 |

### 常用动画配置

```
生命值变化:
- 时长: 300ms
- 曲线: Ease Out Cubic
- 效果: 数字滚动 + 发光脉冲 + 粒子爆发

按钮点击:
- 时长: 150ms
- 曲线: Ease Out
- 效果: 缩放 0.95 + 发光增强

页面转场:
- 时长: 300ms
- 曲线: Ease In Out
- 效果: Fade + Slide

骰子投掷:
- 时长: 1500ms
- 曲线: Elastic Out
- 效果: 3D 旋转 + 反弹

呼吸灯:
- 时长: 2500ms
- 曲线: Ease In Out
- 效果: 透明度 30% ↔ 100% 循环
```

## 🎯 使用快捷方式

### 创建组件快速流程

**1. 创建按钮**:
```
1. 绘制矩形 (120×48, 圆角8)
2. 背景: 透明
3. 边框: 2px, #00D9FF @ 60%
4. 添加发光: Glow/Blue/Medium
5. 添加文本: Button Text Style
6. 创建 Auto Layout (H: 24, V: 12)
7. 转为组件 (Cmd + Option + K)
```

**2. 创建卡片**:
```
1. 绘制矩形 (圆角12)
2. 背景: #FFFFFF @ 6%
3. 边框: 1px, #00D9FF @ 40%
4. 添加发光: Glow/Blue/Soft
5. 创建 Auto Layout (Padding: 16)
6. 转为组件
```

**3. 创建图标**:
```
1. 绘制图标 (24×24)
2. 线宽: 2px
3. 圆角: 处理所有尖角
4. 颜色: #FFFFFF
5. 转为组件
```

## 💡 常见问题速查

**Q: 生命值数字用什么字体？**
A: Roboto Mono Black 96px，字间距 -2px

**Q: 按钮最小点击区域是多少？**
A: 44×44px (iOS 人机界面指南要求)

**Q: 发光效果太强/太弱怎么办？**
A: 使用 Soft (轻微) / Medium (中等) / Strong (强烈) 三个级别

**Q: 如何实现呼吸灯效果？**
A: 透明度在 30%-100% 之间循环，周期 2500ms，Ease In Out

**Q: 粒子大小是多少？**
A: 2px (小), 3px (中), 4px (大)

**Q: 对战界面如何实现对称布局？**
A: Player 2 区域旋转 180°

**Q: 间距不确定用多少？**
A: 默认用 16px (md)，小间距用 8px (xs)，大间距用 24px (lg)
